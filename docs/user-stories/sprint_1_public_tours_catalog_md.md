# Sprint 1 — Public Tours Catalog

## Sprint Goal

Build the public tours catalog and the initial backend foundation for the Tours Booking Platform project.

---

# User Stories Included

## HU-01 · View available tours listing

As a website visitor  
I want to view all available tours on a single page  
so that I can compare options and find a suitable trip.

---

## HU-02 · Filter tours by category

As a visitor  
I want to filter the tours catalog by category  
so that I can quickly find the type of tour I am interested in.

---

## HU-03 · View detailed tour information

As a visitor interested in a tour  
I want to see detailed information about the selected tour  
so that I can decide whether I want to book it.

---
